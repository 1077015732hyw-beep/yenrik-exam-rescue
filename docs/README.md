# yenrik-exam-rescue

个人维护的期末复习资料分享网站。

## Site Purpose

本站用于分享课堂重点、个人整理笔记和 PDF 复习材料。目标用户是同班同学、朋友和学弟学妹。

本站不是学习平台、在线教育网站、题库系统或资源管理后台。

## Deploy Path

所有静态资源路径继续使用 GitHub Pages 项目路径：

```text
/yenrik-exam-rescue/
```

例如：

```text
/yenrik-exam-rescue/css/style.css
/yenrik-exam-rescue/js/app.js
/yenrik-exam-rescue/pdf/advanced-math/final-review.pdf
```

## Pages

- `/`：复习材料首页
- `/pages/advanced-math.html`：高等数学
- `/pages/linear-algebra.html`：线性代数
- `/pages/marxism.html`：马克思主义基本原理
- `/pages/college-english.html`：大学英语
- `/pages/resource.html?id=...`：PDF 在线查看
- `/pages/about.html`：关于与免责声明
- `/pages/acknowledgements.html`：鸣谢
- `/pages/support.html`：支持项目
