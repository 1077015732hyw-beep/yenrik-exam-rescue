关于这座小岛

雁塔提名（Yanta Archive）是一座由大学生长期维护的知识小岛，目前以资料馆（期末复习资料库）为主要内容。

岛上的资料依据课堂重点整理，支持在线翻阅和下载带回。核心理念是：收集 → 提炼 → 排版 → 分享。

免责声明

岛上所有资料均为个人整理，仅供学习参考，不构成任何考试保证。请以任课居民（教师）发布的官方材料为准。如发现资料内容有错漏，欢迎联系反馈，会及时勘误。资料仅供个人学习使用，请勿用于商业用途。

请勿大范围传播。若传播范围超过预期，小岛可能会考虑停止维护甚至关闭。

技术说明

本岛基于 GitHub Pages 部署，使用纯 HTML / CSS / JavaScript 构建，无后端。阅读进度、最近浏览等数据存储在浏览器本地，不上传任何服务器。

本站全由 Vibe Coding 制作——没有手写一行代码，全部通过自然语言对话生成。
